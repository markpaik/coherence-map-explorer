# Data provenance

`data.js` is a verbatim snapshot of the Coherence Map dataset that ships with
achievethecore.org's Coherence Map tool.

- **Source URL:** https://tools.achievethecore.org/coherence-map/data.js
- **Fetched:** 2026-07-15
- **SHA-256:** `8b3949f4f48d8c8159c51bce8e658611781c48d46a17de393815e12426420e19`
- **Modifications:** none. The file is stored exactly as served.

The file assigns one object to `window.cc` containing the math standards
(480), clusters (148), domains (63), directed prerequisite edges (757), and
undirected related-standard edges (283 rows, 142 unique pairs). It also
contains ELA standards and task-collection blocks that serve a different tool
on the site; the build pipeline excludes them.

Public Domain Dedication per https://achievethecore.org/ccpd, with items
marked © excluded from that dedication. The standards text within the data is
from the Common Core State Standards, © 2010 NGA Center for Best Practices and
CCSSO, used under their public license. Credit for the dataset belongs to
Student Achievement Partners.
